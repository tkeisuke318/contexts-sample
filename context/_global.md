# グローバルルール

すべてのフェーズ・タスク種別に共通して適用されるルールです。

## 参照禁止フォルダ

以下のフォルダは、**明示的にメンションされない限り参照してはならない**。

| フォルダ | 理由 |
|---------|------|
| `old_docs/` | 廃版・旧バージョンの成果物。現行の情報源として使用してはならない |
| `context/` | GraphRAG によって自動注入される作業指示。直接参照する必要はない |
| `prompts/` | AI プロンプト定義。成果物生成時の参照対象ではない |

成果物の生成・参照に使用するフォルダは `docs/`（既存成果物）と `templates/`（テンプレート）のみとする。
