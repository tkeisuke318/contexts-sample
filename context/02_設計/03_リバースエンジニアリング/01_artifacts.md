# 設計フェーズ（リバースエンジニアリング）— インプット・アウトプット

## インプット

| # | インプット | 入手先 | 必須/任意 |
|---|-----------|-------|---------|
| 1 | ソースコード | `src/`（リポジトリ） | 必須 |
| 2 | DDL・マイグレーションファイル | `db/`（リポジトリ） | あれば必須 |
| 3 | API 定義ファイル（OpenAPI 等） | `api/`（リポジトリ） | あれば必須 |
| 4 | 設定ファイル（環境変数・設定 YAML 等） | リポジトリルート | あれば必須 |
| 5 | 依存定義ファイル（package.json・pom.xml 等） | リポジトリルート | 必須 |
| 6 | 既存テストコード | `src/test/`（リポジトリ） | あれば必須 |
