## 処理フロー

> ★★ バッチ全体の処理フローをMermaid flowchartで図示する

```mermaid
flowchart TD
    Start([バッチ開始]) --> P0["Step0: 初期処理\n・起動パラメータ検証\n・処理日付取得\n・ログ初期化"]
    P0 --> P1["Step1: データ読み込み\n・対象データをDBから取得\n・取得件数をログ出力"]
    P1 --> Loop{レコードあり？}
    Loop -- なし --> P4
    Loop -- あり --> P2["Step2: バリデーション\n・必須チェック\n・型チェック\n・業務ルールチェック"]
    P2 --> Valid{バリデーションOK？}
    Valid -- NG --> ErrLog["エラーログ出力\n・エラーレコードをスキップ"]
    ErrLog --> Loop
    Valid -- OK --> P3["Step3: メイン処理\n・データ変換・集計\n・DBへ書き込み"]
    P3 --> Loop
    P4["Step4: 終了処理\n・コミット\n・処理件数・エラー件数ログ出力"] --> End([バッチ正常終了])
    P0 -- 例外発生 --> Abort["異常終了処理\n・ロールバック\n・エラーログ出力\n・アラート通知"]
    Abort --> Fail([バッチ異常終了])
```
