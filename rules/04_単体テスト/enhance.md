# 単体テストフェーズ — エンハンス

## 遵守事項

- テストの独立性：各テストケースは他のテストケースの実行結果に依存してはならない
- 外部依存：DB・外部APIはモックに置き換える
- カバレッジ基準：ブランチカバレッジ80%以上

## 禁止事項

- なし

## レビュー観点マーカー

更新箇所は変更内容がわかるようにインラインで記述する。マーカーなしの記述は「変更前から変わっていない」とみなす。

| マーカー | 意味 |
|---------|------|
| `※変更` | 既存記述を変更した箇所 |
| `※追加` | 新たに追記した箇所 |
| `※削除` | 削除した項目（削除理由を添える） |
| `※推測` | 文脈から推論した。根拠はあるが明示されていない |
| `※要確認` | 情報不足で判断できない。仮置きしている |
