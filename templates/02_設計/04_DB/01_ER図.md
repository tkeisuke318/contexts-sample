## ER図（全体）

> ★★ システム全体のテーブル関係をMermaid erDiagramで図示する。全テーブルを網羅すること

```mermaid
erDiagram
    CUSTOMERS {
        bigint id PK
        varchar name
        varchar email
        timestamp created_at
        timestamp updated_at
    }

    ORDERS {
        bigint id PK
        bigint customer_id FK
        date order_date
        varchar status
        timestamp created_at
        timestamp updated_at
        timestamp deleted_at
    }

    ORDER_DETAILS {
        bigint id PK
        bigint order_id FK
        bigint product_id FK
        int quantity
        decimal unit_price
    }

    PRODUCTS {
        bigint id PK
        varchar code
        varchar name
        decimal price
        timestamp created_at
        timestamp updated_at
    }

    CUSTOMERS ||--o{ ORDERS : "1対多"
    ORDERS ||--|{ ORDER_DETAILS : "1対多"
    PRODUCTS ||--o{ ORDER_DETAILS : "1対多"
```
