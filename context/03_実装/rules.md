# 実装フェーズ — 規約

## 規約

コーディング規約の詳細は `docs/03_実装/コーディング規約.md` を参照すること。

- **ブランチ名**：`feature/[チケットID]-[機能名]`（例：`feature/PROJ-123-order-create`）
- **コミットメッセージ**：`[チケットID] 変更内容の概要`
- **マジックナンバー・マジックストリング**：定数またはEnum化する
- **例外処理**：ログを必ず出力してからハンドリングまたは再スローする
- **環境依存値**：URL・パスワード等は環境変数から取得する
