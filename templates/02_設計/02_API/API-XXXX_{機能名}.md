## API詳細定義

> ★★ APIごとにリクエスト・レスポンス（成功時/エラー時）を定義する

### ★★受注登録（POST /orders）

#### リクエスト

| 項目 | 内容 |
|------|------|
| メソッド | POST |
| パス | ★★/orders |
| Content-Type | application/json |

**リクエストボディ**

```json
{
  "★★customer_id": "★★integer（必須）",
  "★★order_date": "★★string（必須）YYYY-MM-DD形式",
  "★★items": [
    {
      "★★product_id": "★★integer（必須）",
      "★★quantity": "★★integer（必須）1以上"
    }
  ]
}
```

| フィールド名 | 型 | 必須 | 制約 | 説明 |
|-----------|---|------|------|------|
| ★★customer_id | integer | 必須 | 1以上 | ★★顧客ID |
| ★★order_date | string | 必須 | YYYY-MM-DD | ★★受注日 |
| ★★items[].product_id | integer | 必須 | 1以上 | ★★商品ID |
| ★★items[].quantity | integer | 必須 | 1以上 | ★★数量 |

#### レスポンス（成功時）

**ステータスコード：201 Created**

```json
{
  "★★id": "★★integer",
  "★★customer_id": "★★integer",
  "★★order_date": "★★string",
  "★★status": "★★string（例：PENDING）",
  "★★items": [],
  "★★created_at": "★★string（ISO8601）"
}
```

#### レスポンス（エラー時）

**共通エラーレスポンス形式**

```json
{
  "error_code": "★★ERR-XXXX",
  "message": "★★ユーザー向けメッセージ",
  "details": [
    {
      "field": "★★エラーフィールド名",
      "message": "★★フィールドレベルのエラーメッセージ"
    }
  ]
}
```

| ステータスコード | エラーコード | 発生条件 |
|--------------|------------|---------|
| 400 | ★★ERR-0001 | ★★バリデーションエラー |
| 401 | ★★ERR-0002 | ★★認証トークン不正・期限切れ |
| 403 | ★★ERR-0003 | ★★アクセス権限なし |
| 404 | ★★ERR-0004 | ★★指定されたリソースが存在しない |
| 500 | ★★ERR-9999 | ★★サーバー内部エラー |
